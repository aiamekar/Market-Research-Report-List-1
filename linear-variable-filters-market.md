<p><p>Linear variable filters are optical devices that selectively transmit specific wavelengths of light while blocking others. In the market, VIS Bandpass filters allow visible light within a defined range, NIR Bandpass targets near-infrared wavelengths, and UV Bandpass filters focus on ultraviolet light. VIS Short Wave Pass filters transmit visible light and block longer wavelengths, while VIS Long Wave Pass filters allow longer visible wavelengths through while blocking shorter ones. Each type serves distinct applications in spectroscopy, imaging, and photometry.</p></p>
